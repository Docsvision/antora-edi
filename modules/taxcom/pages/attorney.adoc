= Настройка работы с МЧД

.Для работы с МЧД через интерфейс {wc}а потребуются опции лицензии:
* При обмене документами через систему {tf} потребуется опция лицензии _Модуль интеграции с реестром МЧД_.
* При передаче МЧД на регистрацию и при отправке МЧД через оператора ЭДО потребуется лицензия _Модуль интеграции с реестром МЧД_ и _{txc}_.

Подробнее про требования для работы с модулем см. в разделе xref:ROOT:requirements.adoc#license[Требования лицензии].
